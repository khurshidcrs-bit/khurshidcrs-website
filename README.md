# Netlify deploy — Khurshid CRS

Dit pakket bevat een kant-en-klare `index.html` die direct werkt op Netlify (of elke andere statische hosting).

## Inhoud
- `index.html` — de volledige site in één bestand
- `netlify.toml` — optioneel, zet basis headers

## Snel deployen op Netlify
1. Ga naar https://app.netlify.com/ en kies **Add new site → Deploy manually**.
2. Sleep de map/zip met deze bestanden erin óf kies **Upload folder** en selecteer de map.
3. Klaar. Je site staat live.

## Aanpassen
Open `index.html` en zoek bovenin naar de sectie **AANPASBARE GEGEVENS**.

- `BEDRIJFSNAAM` (al ingevuld met Khurshid CRS)
- `HEADER_AFBEELDING` — URL van de foto van jullie voorgevel
- `ADRESREGELS` en `CONTACT` (al ingevuld)
- `VOORRAAD` — lijst met auto’s
